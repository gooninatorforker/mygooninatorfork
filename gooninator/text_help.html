The Gooninator's Text Feature

Text comes from scripts defined by Pastebins. Here's an example: 

setBlinkDuration 300
setBlinkDelay 100
setBlinkGroupDelay 1200
setCaptionDuration 2000
setCaptionDelay 1200

bigcap YOU LOVE FLUFFY KITTENS
blink KITTENS / ARE / YOUR / LIFE
cap Cuddle all the kittens forever because you love them.

In the example above, the following things are true:

  • A "caption" will be displayed for 2000ms (2 seconds).
  • After a "caption" disappears, there will be a 1200ms (1.2 second) delay.
  • A "blink" will split the text on slashes and show the segments one at a time, each for 300ms, followed by 100ms of nothing. So you'd see KITTENS for 300ms, then nothing for 100ms, then ARE for 300ms, etc.
  • After each group of blinks (a single line), there will be a 1200ms delay.
  • The script will repeat.

Other supported commands:

  • wait 1000 waits for 1 second
  • storePhrase foo bar stores "foo bar" in a list of phrases. Use the text $RANDOM_PHRASE to use a random phrase from the list. For example, blink $RANDOM_PHRASE. example

There are some raw CSS commands, but I don't have the energy to document them right now and I don't really expect people to use them.

The result is dumped into a Pastebin, and the ID of that Pastebin can be put in the "text pastebin ID" field on the main page of the Gooninator.
